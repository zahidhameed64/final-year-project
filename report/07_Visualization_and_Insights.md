# Chapter 7: Visualization and Insights

## 7.1 Dashboard Design

The web application's dashboard serves as the primary interface for "Data Storytelling."

*   **Interactive Charts:** Utilized `Recharts` to move beyond static images. Users can hover over the "Feature Importance" bar chart to see precise coefficient values.
*   **Real-Time Feedback:** The "Predicted Earnings" card updates instantly, providing a "Calculator" experience rather than a static report.

## 7.3 Summary of Key Insights

1.  **Engagement > Reach:** 100k views are worth more than 10k subscribers.
2.  **Niche Matters:** A finance channel needs 1/10th the traffic of a prank channel to make the same revenue.
3.  **Global Disparity:** Location is a massive multiplier for ad rates.
